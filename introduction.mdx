Equinox is a native macOS app that lets you create dynamic wallpapers — the kind macOS has supported since Mojave. Instead of a static image sitting on your desktop, a dynamic wallpaper responds to the world around it: it can follow the arc of the sun across the sky, cycle through images on a schedule you set, or simply flip between a light and dark version as you switch system appearance. Equinox gives you a straightforward interface for building any of these wallpaper types from your own photos, without writing a single line of code.

## Wallpaper types

Equinox supports the three dynamic wallpaper formats that macOS provides natively. Each type has a different trigger — the sun's position, the time of day, or the current system appearance — so you can choose the one that best fits your photos.

## Solar Calculator

When creating a Solar wallpaper, each image needs to be tagged with the sun's altitude and azimuth at the moment it was taken. Equinox includes a built-in Solar Calculator to help you find those values without any external tools. You enter the location, date, and time for a photo, and the calculator determines the sun's position. You can then drag and drop the result directly onto the image in your wallpaper.

<Tip>
  If you don't know the exact time a photo was taken, use the Sun timeline in the Solar Calculator to visually match the sun's height to what you see in the photo.
</Tip>

## Requirements and availability

Equinox requires **macOS 10.14 Mojave or later**. Dynamic wallpaper support was introduced in Mojave, so older versions of macOS are not supported.

You can download Equinox from two sources:

- **Mac App Store** — the easiest way to install and receive updates automatically.
- **GitHub Releases** — open-source builds available at [github.com/rlxone/Equinox](https://github.com/rlxone/Equinox/releases).
