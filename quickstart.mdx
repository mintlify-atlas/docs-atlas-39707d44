---
title: "Quickstart: Create Your First Dynamic Wallpaper in Equinox"
description: "Build an Appearance wallpaper in Equinox using two images — one for Light Mode, one for Dark Mode — and set it as your desktop background."
---

The fastest way to get started with Equinox is to create an Appearance wallpaper. This type only requires two images — one for Light Mode and one for Dark Mode — and macOS switches between them automatically whenever you change your system appearance. There is no scheduling or sun-position data to configure, which makes it the ideal starting point before moving on to Time or Solar wallpapers.

## Supported image formats

Equinox accepts images in four formats: **PNG**, **JPEG**, **TIFF**, and **HEIC**. Any images you plan to use should be in one of these formats before you begin. Both images in an Appearance wallpaper do not need to share the same format.

<Tip>
  For the best results, use images with the same resolution and aspect ratio. macOS crops wallpapers to fit the screen, so mismatched dimensions can produce unexpected framing.
</Tip>

## Create an Appearance wallpaper

<Steps>
  <Step title="Open Equinox">
    Launch Equinox from Launchpad or your Applications folder. If this is your first time opening the app, complete the introductory walkthrough before continuing.
  </Step>
  <Step title="Select the Appearance wallpaper type">
    On the wallpaper type selection screen, choose **Appearance**. This type creates a wallpaper with one image for Light Mode and one for Dark Mode.
  </Step>
  <Step title="Add your Light Mode image">
    In the Light slot, click the image well or drag and drop your Light Mode image onto it. Equinox accepts PNG, JPEG, TIFF, and HEIC files.
  </Step>
  <Step title="Add your Dark Mode image">
    In the Dark slot, click the image well or drag and drop your Dark Mode image onto it. This image displays whenever macOS is in Dark Mode.
  </Step>
  <Step title="Create the wallpaper">
    Click **Create**. Choose a save location when prompted. Equinox exports a single `.heic` file that contains both images encoded as a macOS dynamic wallpaper.
  </Step>
  <Step title="Set the wallpaper as your desktop background">
    In Finder, locate the exported `.heic` file. Right-click it and choose **Services → Set Desktop Picture**. macOS will apply it immediately.

    <Note>
      If **Set Desktop Picture** does not appear in the Services submenu, you may need to enable it in **System Settings → Privacy & Security → Extensions → Finder Extensions**.
    </Note>
  </Step>
</Steps>

## Verify the wallpaper works

After setting the wallpaper, switch your system appearance between Light and Dark Mode in **System Settings → Appearance** to confirm both images display correctly.

<Info>
  If the wallpaper does not switch when you change appearance, check that your desktop is not set to a static wallpaper. Open **System Settings → Wallpaper**, confirm Equinox's wallpaper is selected, and switch appearance again.
</Info>
